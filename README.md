# Spring Boot + Angular + Ant Design Archetype

## Usage

```
mvn archetype:generate \
    -DarchetypeGroupId=com.luhuiguo.archetypes \
    -DarchetypeArtifactId=spring-angular-antd-archetype \
    -DarchetypeVersion=0.3.0
```
